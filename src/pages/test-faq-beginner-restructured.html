🔴 지금 막혔어요! (즉시 해결 필요)

설치 중 진행이 막혔다면 여기서 해결책을 찾으세요.

🔐 비밀번호를 입력하는데 화면에 안 나와요!

정상입니다! 보안상 비밀번호가 화면에 표시되지 않는 것이 맞습니다.

✅ 해결법

  1. 비밀번호 입력: 화면에 안 보여도 입력되고 있어요
  2. Enter 누르기: 다 입력했으면 Enter
  3. 틀렸다면: "Sorry, try again." 나오면 다시 입력

🚫 Permission denied 오류가 나요

프로그램을 전역으로 설치하려면 관리자 권한이 필요합니다.

✅ Mac 해결법: sudo 사용

sudo npm install -g @anthropic-ai/claude-code

비밀번호 입력 후 Enter

✅ Windows 해결법: 관리자 권한으로 실행

  1. Git Bash 닫기
  2. Git Bash 우클릭 → "관리자 권한으로 실행"
  3. 명령어 다시 실행

📋 명령어가 너무 길어서 복사-붙여넣기가 안 돼요

🍎 Mac 복사-붙여넣기

  1. 명령어 선택 후 ⌘ Command + C
  2. 터미널 클릭 후 ⌘ Command + V
  3. Enter 누르기

🪟 Windows 복사-붙여넣기

  1. 명령어 선택 후 Ctrl + C
  2. Git Bash에서 Ctrl + Shift + V
  3. 또는 우클릭 → "Paste"
  4. Enter 누르기

🌐 브라우저가 안 열려요

✅ 수동으로 열기

  1. 터미널에서 https://로 시작하는 URL 찾기
  2. URL 전체 복사
  3. 브라우저 직접 열어서 주소창에 붙여넣기
  4. Claude 계정으로 로그인

🤖 claude-code 실행했는데 반응이 없어요

몇 초 기다려보세요. 처음 실행 시 시간이 걸릴 수 있습니다.

🔧 문제해결 순서

  1. 설치 확인: claude-code --version
  2. 로그인 확인: claude-code auth status
  3. 재로그인: claude-code auth login
  4. 터미널 재시작

🟡 이게 정상인가요? (불안함 해소)

정상적인 과정이니 걱정하지 마세요.

😰 영어로 뭔가 물어보는데 뭘 선택해야 하나요?

✅ 황금 규칙

  • "Continue?" → Y 입력
  • "Install?" → Y 입력
  • "Agree?" → Y 입력
  • "Press ENTER" → 엔터 누르기

🎯 핵심: 거의 모든 질문에 "예(Y)"라고 답하면 됩니다!

⏰ 설치가 5분째 진행 중인데 멈춘 건가요?

😌 정상입니다!

  • 소요시간: 5-10분 정도 걸릴 수 있어요
  • 진행 확인: 글자가 계속 바뀌면 진행 중
  • 스피너: ⠸⠹⠼⠽ 이런 기호가 돌면 작동 중

💡 커피 한 잔 마시고 오세요!

✅ 로그인했는데 다음에 뭘 해야 하나요?

✅ 정상적인 흐름

  1. 브라우저: "로그인 성공" 메시지
  2. 터미널로 돌아가서 확인
  3. "Successfully logged in" 메시지 확인
  4. 브라우저 탭 닫기
  5. claude-code 실행해서 사용 시작!
